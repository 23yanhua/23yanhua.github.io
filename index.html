ftp服务器搭建（Linux）

相关软件 vsftpd

如果是第一次使用，在命令行输入 yum install vsftpd

服务器设置

1.服务器环境设置

服务器启动模式:listen=YES(默认为stand daemon，如果改用super daemon应启动xined，listen=NO）
欢迎信息
配置banner_file=/etc/vsftpd/welcome.txt
建立欢迎界面
vim /etc/vsftpd/welcome.txt

2.与实体用户较相关设置

3.匿名用户登录设置

匿名用户只能在/var/ftp浏览（可在此目录放置文件）

4.关于系统安全方面设置

==防火墙加入所需要的FTP模块==
vim /etc/sysconfig/iptables-config
IPTABLES_MODULES=”ip_nat_ftp ip_conntrack_ftp”
开启iptables /etc/init.d/iptables restart

==兼顾主动式与被动式连接==
iptables -A INPUT -p TCP -i eth0 –dport 21 –sport 1024:65534 -j ACCEPT
iptables -A INPUT -p TCP -i eth0 –dport 65400:65410 –sport 1024:65534 -j ACCEPT

附：
ftp账号添加命令：useradd passwd
知道自己ftp服务器的ip地址： ifconfig

登录ftp
ftp –> open [服务器ip地址]

存在问题

默认一般用户登录无法取得用户主目录，是SElinux问题，可通过修改规则：
setsebool -P ftp_home_dir=1
